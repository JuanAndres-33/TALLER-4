## 📌 Descripción del proyecto

Este proyecto consiste en el desarrollo de una aplicación en Java que simula la gestión de un mundial utilizando **matrices y arreglos**.

El programa permite:

- Mostrar banderas de países en consola usando **pixel art** y matrices.
- Visualizar una **tabla de posiciones** editable.
- Consultar el **fixture de partidos**.
- Obtener información de cada país (capital, jugadores, etc.).
- Navegar mediante un **menú en consola con ASCII art**.

El objetivo es aplicar conceptos de:
- Ciclos
- Arreglos
- Matrices
- Entrada por teclado
- Organización de datos
- Trabajo colaborativo

---

## ⚙️ Funcionalidades

### 🏳️ Visualización de banderas
- Uso de matrices para representar banderas.
- 4 tamaños:
  - Grande
  - Mediano
  - Pequeño
  - Miniatura (ícono)

---

### 📊 Tabla de posiciones
Se implementa una matriz con los siguientes datos:

- PJ (Partidos Jugados)
- PG (Partidos Ganados)
- PE (Empates)
- PP (Perdidos)
- GF (Goles a Favor)
- GC (Goles en Contra)
- DG (Diferencia de Gol)
- TA (Tarjetas Amarillas)
- TR (Tarjetas Rojas)
- Pts (Puntos)

✔ Se imprime en consola de forma organizada  
✔ Puede actualizarse dinámicamente  

---

### 📅 Fixture de partidos
- Uso de arreglos y matrices
- Permite:
  - Ver partidos por grupo
  - Consultar detalles de un partido

---

### 🌍 Información de países
Permite consultar:

- Capital
- Jugadores principales
- Participaciones en mundiales
- Bandera del país

---

### 🎨 Interfaz en consola
- Menú interactivo
- Uso de **ASCII art**
- Navegación por teclado
